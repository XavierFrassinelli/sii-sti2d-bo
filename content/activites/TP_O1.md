# Activités et TP - O1 : Caractériser

## TP 1.1 - Analyse d'un produit : Vélo électrique

**Niveau :** Débutant | **Durée :** 2h | **Format :** Groupe (3-4 élèves)

**Objectifs pédagogiques :**
- Identifier les constituants d'un vélo électrique
- Tracer les flux d'énergie (électrique → mécanique)
- Justifier les choix structurels (matériaux batterie, moteur)

**Matériel :**
- Vélo électrique démontable (ou photos détaillées)
- Fiches techniques (batterie, moteur)
- Questionnaire d'analyse (fourni)

**Déroulement :**
1. **Exploration** (20 min) : élèves démontent/observent vélo, photo constituants clés
2. **Schéma bloc** (30 min) : tracent flux énergie + identifient 10-15 constituants majeurs
3. **Analyse développement durable** (40 min) : 
   - Batterie Li-ion : densité énergétique vs poids, cycle charge, recyclage
   - Cadre aluminium : propriétés, impact environnemental vs acier
   - Moteur brushless : rendement, durée vie vs moteur classique
4. **Restitution** (30 min) : présentation A3 schéma + tableau justifications

**Livrables :**
- Schéma bloc annoté
- Fiche justification 3 constituants (matériau, fonction, impact DD)
- Présentation oral 5 min

**Évaluation :** Complétude schéma (CO1.1), justification (CO1.1), approche DD (CO1.3)

## TP 1.2 - Test d'ergonomie : Interface d'application mobile

**Niveau :** Intermédiaire | **Durée :** 3h | **Format :** Binôme

**Objectifs pédagogiques :**
- Analyser l'ergonomie d'une interface (smartphone)
- Identifier défauts de design et accessibilité
- Proposer améliorations justifiées

**Matériel :**
- Smartphone/tablette + application (ex: gestion maison, photo, communication)
- Questionnaire SUS (System Usability Scale) simplifié
- Grille d'analyse ergonomique (10 critères)

**Déroulement :**
1. **Briefing utilisateur** (20 min) : l'un joue utilisateur novice, l'autre observe
2. **Tâche à accomplir** (30 min) : 5-6 tâches précises (ex: "Change paramètre couleur fond")
3. **Observation** (20 min) : noter temps, erreurs, hésitations, commentaires
4. **Questionnaire** (20 min) : utilisateur complète SUS (8-10 questions)
5. **Analyse** (50 min) : binôme synthétise résultats, identifie 5 problèmes ergonomie
6. **Propositions** (30 min) : sketcher 3 améliorations avec justification

**Livrables :**
- Fiches observation + chrono
- Résultats questionnaire SUS
- Rapport 2-3 pages : problèmes identifiés + propositions
- Sketches améliorations

**Évaluation :** Analyse ergonomique (CO1.2), propositions (CO1.2), justification (CO1.2)

## TP 1.3 - Calcul d'impact environnemental simplifié

**Niveau :** Avancé | **Durée :** 4h | **Format :** Groupe (2-3 élèves)

**Objectifs pédagogiques :**
- Comparer impacts environnementaux 2-3 solutions
- Utiliser une ACV (Analyse Cycle de Vie) simplifiée
- Justifier choix constructif selon impact

**Matériel :**
- Fichier Excel ACV pré-rempli (template fourni)
- Datasheets produits/matériaux
- Base données INIES ou EcoInvent (accès libre online)

**Cas d'étude proposé :** Comparaison emballage biodégradable vs carton recyclé vs plastique PET

**Déroulement :**
1. **Collecte données** (60 min) : cherchent données (masse, matériau, proc. fabri., transport)
2. **Calcul ACV simplifiée** (60 min) : remplissent Excel pour 3 critères (fabri., transport, EOL)
   - Émissions CO2 équivalent
   - Énergie consommée
   - Potentiel recyclage
3. **Graphiques comparatifs** (40 min) : tracent histogrammes, analyse résultats
4. **Rapport recommandation** (40 min) : choisissent meilleure option, justifient

**Livrables :**
- Fichier Excel complété
- Graphiques comparatifs (3-4 histogrammes)
- Rapport 2-3 pages : données → résultats → conclusion

**Évaluation :** Calcul impact (CO1.3), justification (CO1.3), lien efficacité (CO1.3)
